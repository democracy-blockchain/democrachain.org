FAQ

What is a Smart-contract?
A smart contract is a little piece of code (written in javascript) that is written inside a block and adds unique features and functionalities to the network, always transparent and understandable. In other words, a smart contract acts an unbreakable agreement between parties capable of executing and enforcing itself through the blockchain system. The two main characteristics of a smart contract are: • It is immutable, which means that, thanks to the fact that it operates under Blockchain technology, once it has been signed on the blockchain, it can no longer be edited or modified. • It is decentralized. It does not depend on a single entity.
What is a block? It's a plain text file that has to comply with certain requirements to be accepted by the network. For example it requires the hash of the latest block on it's scope, and the signature of who created it.
What is a node? All blocks are stored and processed by a network of nodes that execute the same program, allowing the network to work without a single point of failure. Democrachain nodes can run inside AWS Lambda functions, making it the most cost-effective way to run a blockchain network.
What is a digital signature? It's a piece of text that is generated using a private key and can be verified using a public key. Making it the digital version of a real-world signature.
